算法题：

给定N个用户的朋友列表。对于一个指定的用户，找出该用户最可能认识的人。最可能认识的人的定义是：这个人和当前用户不是朋友，但是他们有最多的共同朋友。朋友关系是相互的

例子：

5 0

1 2 3

2 4

1 3

0 1

1 2 3

上面第一行5表示一共5个人，指定用户为0，后面跟着5行为5个人的好友列表。
